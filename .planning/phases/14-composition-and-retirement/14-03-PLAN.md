    ```

    ```

    For commands with hyphens: `state advance-plan` becomes `state.advance-plan`.

    **Pattern C — Fire-and-forget mutations:**

    ```
    BEFORE: node "$HOME/.claude/get-shit-done/bin/gsd-tools.cjs" state advance-plan

    ```
