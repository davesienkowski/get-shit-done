    `writeStateMd(statePath, content, projectDir)`: Calls syncStateFrontmatter, acquires lock, writes normalizeMd(synced), releases lock.

    `readModifyWriteStateMd(projectDir, modifier)`: Acquires lock on STATE.md path, reads file, applies async modifier function, syncs frontmatter, writes, releases lock. The modifier signature is `(content: string) => Promise<string> | string`.

    Port all 10 command handlers as QueryHandlers:

    `stateUpdate`: args[0]=field, args[1]=value. Uses readModifyWriteStateMd with stateReplaceField. Returns `{ data: { updated: true, field, value } }`. Throws GSDError(Validation) if field/value missing. Returns `{ data: { updated: false, field } }` if field not found.

    `stateAddBlocker`: args[0]=blocker text. Appends `- {text}` under `## Blockers` section.

    `stateResolveBlocker`: args[0]=search text. Removes the first line under Blockers matching the search text.

    `stateRecordSession`: args[0]=timestamp, args[1]=stoppedAt, args[2]=resumeFile (optional). Updates "Last session", "Stopped at", "Resume file" fields.
